ThreadDispatcherShutdownMode Enumeration

Describes the current shutdown mode of a thread dispatcher.

Namespace:  RI.Abstractions.Dispatcher
Assembly:  RI.Abstractions.Common (in RI.Abstractions.Common.dll) Version: 1.4.0.0

[SerializableAttribute]
public enum ThreadDispatcherShutdownMode
Members
  Member nameValueDescription
None0 The dispatcher is not running or currently not being shut down.
DiscardPending1 The dispatcher is being shut down and all already pending delegates are discarded.
FinishPending2 The dispatcher is being shut down and all already pending delegates are processed before the shutdown completes.
AllowNew3 The dispatcher is being shut down and, already pending delegates are processed before the shutdown completes, and new delegates can still be enqueued.
