Remarks
 Important
SimpleContainer does not provide the same functionality as full-fledged dependency injection libraries. As the name suggests, it provides only simple composition container and dependency injection functionality.
 Important
SimpleContainer only provides the following functionality: Support for IEnumerableT to get multiple instances; constructors with or without parameters.
 Important
Per SimpleContainer instance, Register(IEnumerableCompositionRegistration) can only be called once and GetService(Type) and GetServices(Type) can only be called after Register(IEnumerableCompositionRegistration).
 Note
If multiple constructors are available, the one of which the most parameters can be resolved will be used (best match). If there is no best match constructor, an exception is thrown. If there are multiple best match constructors, an exception is thrown.
 Note
SimpleContainer is used internally by BuildStandalone(IBuilder).
 Note
Temporary registrations are ignored.
Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
