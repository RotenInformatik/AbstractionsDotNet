IThreadDispatcherOperationWait Method

Waits a specified amount of time for the thread dispatcher operation to finish processing.

Namespace:  RI.Abstractions.Dispatcher
Assembly:  RI.Abstractions.Common (in RI.Abstractions.Common.dll) Version: 1.4.0.0
Syntax
C#
Copy
bool Wait(
	TimeSpan timeout,
	CancellationToken cancellationToken
)

Parameters

timeout
Type: SystemTimeSpan
The maximum time to wait for the thread dispatcher operation to finish processing before the method returns.
cancellationToken
Type: System.ThreadingCancellationToken
The cancellation token which can be used to cancel the wait.

Return Value

Type: Boolean
true if the thread dispatcher operation finished processing within the specified timeout, false otherwise or if the wait was cancelled.
