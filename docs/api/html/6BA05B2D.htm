Contains contracts and types to abstract delegate processing using a prioritized queue.

Interfaces
  InterfaceDescription
IThreadDispatcher
Standalone thread-bound dispatcher abstraction.
IThreadDispatcherOperation
An operation (delegate) scheduled to be executed by a thread dispatcher.
IThreadDispatcherRunnable
An object which can be executed by a thread dispatcher.
IThreadDispatcherTimer
An operation (delegate) scheduled to be executed by a thread dispatcher either delayed (one-shot) or in repeated intervals (continuous).
