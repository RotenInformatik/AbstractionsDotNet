Remarks
 Notes to Implementers
IThreadDispatcherTimer enqueues a delegate to the specified dispatchers queue (using Post(ThreadDispatcherExecutionContext, Int32, ThreadDispatcherOptions, Delegate, Object)) in a specified interval (never before).
 Notes to Implementers
The interval is awaited before the timer is executed for the first time. Afterwards, the delegate is posted to the dispatcher in the specified interval.
 Notes to Implementers
If a previous interval has not yet finished executing the delegate, it is not dispatched for execution until the next interval.
 Notes to Implementers
The timer is initially stopped and needs to be started explicitly using Start(TimeSpan).
 Notes to Implementers
The associated thread dispatcher does not need to be started. However, the timer is stopped on an interval if the dispatcher is not running.
