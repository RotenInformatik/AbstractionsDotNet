Remarks
 Important
When building (calling Build or BuildStandalone(IBuilder)), BuilderBase expects exactly one instance of ILogger, one instance of ICompositionContainer, and zero instances of IBuilder already registered.

BuilderBase adds itself as a Temporary registration with IBuilder as the contract.

During Build or BuildStandalone(IBuilder), after PrepareRegistrations(ILogger) but before PerformRegistrations(ILogger, ICompositionContainer) is called, all Temporary, ILogger, and ICompositionContainer registrations will be removed.

Thread Safety
Static members of this type are not safe for multi-threaded operations. Instance members of this type are not safe for multi-threaded operations.
