Remarks

ICompositionContainer is used to abstract away the used composition, container and/or dependency injection library and mechanism.

ICompositionContainer is intended to provide a mechanism (through Register(IEnumerableCompositionRegistration)) for one-time registration of services (e.g. by a IBuilder implementation during the build stage)
